# FileFlow - Convert, Compress & Share

A modern web application for converting, compressing, and sharing files via personal web links — no contact exchange needed.

## Features

- **File Conversion** - Convert between text, documents, and various file formats
- **Compression** - Reduce file sizes while maintaining quality
- **Instant Sharing** - Share files via personal web links without needing contacts
- **No Sign-up Required** - Recipients can access shared files directly

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI)
- React Router

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
